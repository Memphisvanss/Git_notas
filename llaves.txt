Notas llaves publicas y privadas

ruta: /Users/ivangs/.ssh

Descripcion:
Las llaves publicas y privadas sirven para compartir mensajes encriptados por algun medio no seguro

Funcionamiento:
El usuario que quiere recivir mensajes crea su llave publica y su llave privadas
la llave publica la comparte con el usuario de quien quiere recibir los mensajes
el otro usuario B encripta el mensaje con la llave publica del usuario A y envia el mensaje
el usuario A puede desencriptar el mensaje usando su llave privada.

La llaves se crean al mismo tiempo por un metodo matematico pero la llave publica 
no sirve para decifrar el mensaje por lo que se pueden compartir por cualquier medio
publico si arriesgar el mensaje.

MAC
Generar llaves 
ssh-keygen -t rsa -b 4096 -C "youremail@example.com"

Revisar que agente de ssh esta corriendo 
eval "$(ssh-agent -s)"

Crear el archivo config en el directorio .ssh
vim config

el archivo debe de contener:
Host *
  AddKeysToAgent yes
  UseKeychain yes
  IdentityFile ~/.ssh/id_rsa

Agregar la llave privada 
ssh-add -K ~/.ssh/id_rsa
